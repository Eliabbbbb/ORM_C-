Proyecto: Gestor de Consultas "NorthWind"

Este informe presenta un conjunto de consultas de base de datos desarrolladas en C# utilizando un ORM (Object-Relational Mapping). El proyecto, realizado con Visual Studio 2022 sobre la base de datos "NorthWind", tiene como objetivo representar la información de manera estructurada y orientada a objetos.

Tipos de Consultas Implementadas

1. Consultas de Selección (Filtrado de Filas)

Se centran en seleccionar y filtrar filas específicas de la base de datos.

Clientes por País

Muestra solo los clientes que viven en México.

Proveedores por País

Muestra los proveedores ubicados en Estados Unidos (USA).

Productos por Stock

Muestra productos con existencias (stock) menores a 20.

2. Consultas de Proyección (Selección de Columnas)

Seleccionan únicamente columnas (propiedades) específicas de una tabla.

Producto - Precio

Proyecta solo el código y el precio del producto.

Empleado - Origen

Selecciona el código del empleado y su país de origen.

Categorías - Descripciones

Extrae el nombre de la categoría y su descripción.

Desarrollo y Tecnologías

El proyecto fue implementado en ASP.NET (C#), utilizando las herramientas de Visual Studio para conectar el modelo de datos (ORM) con la interfaz de usuario.

Hosting: Tanto la aplicación web como la base de datos están alojadas en la plataforma Somee.com.

Tecnologías Utilizadas:

  • ORM en C#
  • Visual Studio 2022
  • ASP.NET
  • Base de Datos "NorthWind" (SQL Server)
  • Hosting: Somee.com
